lo que aprendide hoy
git init: sirve para inicializar git, siempre que quiera clonar un nuevo proyecto o subir un proyecto hay que inicializar git
git add .:sirvepara indicar que todos mis archivos nuevos o con cambios se suben 
git commit -m "describe los cambios":sirve para describir los nuevos cambios de una manera breve
git branch -M main: se usa para indicar a la rama que quiero subir al proyecto, por defeccto es main pero puedo crear nuevas
git push -u origin main: sirve para subir los archivos nuevos o modificados a mi repositorio de github
